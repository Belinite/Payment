Сверстать форму оплаты со следующими полями:

- Тип плательщика: выпадающий список. Допустимые значения:
	"Физическое лицо"
	"Юридическое лицо"
- НДС - текстовое поле, заблокировано. Если тип плательщика выбран "Физ. лицо", то в поле устанавливается значение "13". Если "Юр. лицо" - "17".
- Имя плательщика - текстовое поле. Обязательно к заполнению
- форма ЮЛ - выпадающий список. Поле видно только в случае, если тип плательщика = "Юр. лицо". Допустимые значения:
	"ООО"
	"ОАО"
	"ЗАО"
	"ИП"
- ИНН - текстовое поле, обязательно к заполнению. Должна происходить валидация введенных данных: 
	для типа плательщика = "Юр лицо" допустимое значение - 10 цифр
	для типа плательщика = "физ лицо" допустимое значение - 12 цифр
- упрощенное налогооблажение - чексбокс. Поле видно только в случае, если тип плательщика = "Юр. лицо".
	Если установлена галочка, то поле "ИНН" очищается, делается необязательным и недоступным для ввода данных.
- Контактный телефон - текстовое поле. Позволяет ввести значения вида +7хххххххххх, где х - цифры от 0 до 9.
- Сумма платежа - текстовое поле, обязательно для заполнения. Валидация: позволяет ввести положительное число с плавающей точкой, отделитель десятичной части - "точка".
	Если введено корректное число, то оно приводится к виду десятичного числа с 2-мя знаками после разделителя
- Итог - текстовое поле, заблокировано. Если в поле "сумма платежа" введено корректное значение,
	то поле "итог" = "Сумма платежа" * (1 + "НДС"/100).
	Если в поле "сумма платежа" введено НЕкорректное значение, то поле "итог" очищается.
